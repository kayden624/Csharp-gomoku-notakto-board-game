# C# Gomoku & Notakto Board Game

A modular board game framework developed in C# using .NET 8.  
This project supports multiple game modes including Gomoku and Notakto, with both Player vs Player and Player vs Computer gameplay.

---

## Features

- Gomoku game mode
- Notakto game mode
- Player vs Player (PvP)
- Player vs Computer (PvC)
- Move handling system
- Rule validation
- Command system
- Object-oriented architecture
- Modular game framework design

---

## Technologies Used

- C#
- .NET 8
- Object-Oriented Programming (OOP)
- Console Application Development

---

## Project Structure

```txt
Board.cs              -> Board logic
CommandSystem.cs      -> Command processing
GameSystem.cs         -> Main game controller
HelpSystem.cs         -> Help and instructions
Move.cs               -> Move handling
Player.cs             -> Player data and behaviour
Program.cs            -> Entry point
Rule.cs               -> Game rule validation
```
---


## How to Run

### 1. Clone the repository

```bash
git clone https://github.com/kayden624/Csharp-gomoku-notakto-board-game.git
```

---

### 2. Open the project

Open:

```txt
BoardGame.sln
```

using Visual Studio or VS Code.

---

### 3. Run the project

Using terminal:

```bash
dotnet run
```
